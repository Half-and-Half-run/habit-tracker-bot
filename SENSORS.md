# 💡 物理センサーを使った「完全自動記録」のやり方

このBotの最大の強みは、「どんな方法でも `/checkin` というURLにデータさえ送れば記録される」という点です。

つまり、ご自身でスマホをポチポチと押す（自己申告）必要はなく、**「ドアが開いた」「カメラが人を検知した」などの物理的なアクションをトリガーにして、全自動で記録（Webhook送信）させることが可能です。**

以下に、初心者でも簡単に実現できるいくつかの方法をご紹介します。

---

## 🚪 方法1：SwitchBot（スイッチボット）を使う

スマートホーム機器で有名な「SwitchBot」のセンサーを使えば、もっとも簡単に物理世界とBotを連携できます。

### 必要なもの
* **SwitchBot 開閉センサー** または **人感センサー**
* **SwitchBot ハブミニ / ハブ2**（センサーをインターネットに繋ぐため）

### やり方（シーン機能を使ったWebhookの送信）
1. SwitchBotアプリを開き、「シーン」タブから新しくオートメーションを作成します。
2. **【条件（トリガー）】**
   * 例：「開閉センサー」が「開いた」とき。
   * または「人感センサー」が「動体を検知した」ときなど。
3. **【アクション】**
   * 「アクションを追加」から「**Webhook**」を選択します。（※ハブが必要です）
   * **URL:** ngrok等で発行したBotの公開URL（例: `https://xxxx.ngrok-free.app/checkin`）
   * **Method:** `POST`
   * **HTTP Header (JSON形式):** 空欄または `{"Content-Type": "application/json"}`
   * **Body (JSON形式):** `{"action": "wake"}` または `{"action": "bath"}`
4. 保存して完了！
   * これで、朝に「寝室のドアを開けた」だけで自動的に起きたことになり、「お風呂のドアを開けた」だけで入浴したとBotに報告されます。

## 🌐 方法2：IFTTT（イフト）を使って他の機器と連携する

IFTTTは「もし〇〇したら、△△する」というルールを簡単に作れるサービスです。様々なスマート家電（Amazon Alexa、Google Home、Philips Hueなど）とBotを繋げます。

### やり方
1. IFTTT（[https://ifttt.com/](https://ifttt.com/)）に登録します。
2. 新しいAppletを作成します。
3. **【If This（条件）】**
   * 「Alexaで『アレクサ、おはよう』と言ったら」
   * 「スマートプラグの電源がオンになったら」
   * 「スマート電球が点灯したら」
   など、お持ちのスマート家電の動作を選びます。
4. **【Then That（結果）】**
   * サービスから「**Webhooks**」を選び、「Make a web request」を選択します。
   * **URL:** `https://xxxx.ngrok-free.app/checkin`
   * **Method:** `POST`
   * **Content Type:** `application/json`
   * **Body:** `{"action": "wake"}`
5. 保存！
   * これで「アレクサにおはようと言う＝Botに起床記録を送信」という完全自動システムが完成します。

## 📱 方法3：NFCタグ（かざすだけシール）を使う

1枚100円前後で買える「NFCタグ」というシールをスマホでタッチするだけで、自動で通信させる方法です。（iPhone/Androidどちらも対応）

### やり方（iPhoneの場合）
1. Amazonなどで「NFCタグ」を購入し、洗面所やベッドの横など、必ず行く場所に貼り付けます。
2. iPhoneの「ショートカット」アプリを開き、「オートメーション」タブから「個人用オートメーションを作成」を選択します。
3. **【条件】**
   * 「NFC」を選び、「スキャン」をタップして、貼ったNFCタグにiPhoneをかざします。名前（例：洗面所用）を付けます。
4. **【アクション】**
   * アクションを追加で「URLの内容を取得」を選びます。
   * **URL:** `https://xxxx.ngrok-free.app/checkin`
   * **方法:** `POST`
   * **ヘッダ:** `Content-Type` を `application/json` に設定。
   * **本文を要求:** 「JSON」を選び、キーに `action`、テキストに `wake` を追加。
5. 「実行の前に尋ねる」をオフにして保存。
   * これで、朝起きて「洗面所に貼ってあるシールにスマホをポンッとタッチするだけ」でBotに記録が送信されるようになります！

## 🎯 IoTで失敗を防ごう！
Botの `/checkin` エンドポイントが「JSON形式で `action` を受け取る」という非常にシンプルな設計になっているため、世の中のほとんどのスマートホーム機器や自動化アプリからデータを送りつけることができます。自己申告が面倒だと感じる方は、ぜひこれらの「物理連携」を試してみてください！
